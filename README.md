# Portfólio — Célio Bastian

Site pessoal responsivo para apresentar perfil, habilidades e formas de contato.

## Visual
Abra [index.html](index.html) no navegador para ver o projeto.

## Funcionalidades principais
- Navegação com scroll suave via [`scrollToSection`](script.js).
- Destaque do link ativo durante o scroll via [`highlightActiveNavLink`](script.js).
- Animações de entrada usando Intersection Observer via [`animateOnLoad`](script.js).
- Copiar email/contato via área de transferência com [`copyToClipboard`](script.js).

## Tecnologias
- HTML5 — [index.html](index.html)  
- CSS3 — [style.css](style.css)  
- JavaScript (Vanilla) — [script.js](script.js)

## Como rodar (local)
1. Clonar este repositório ou baixar os arquivos.
2. Abrir o arquivo [index.html](index.html) no navegador.
   - Alternativa: usar a extensão Live Server do VS Code para recarregamento automático.
